# [fq.nz](https://fq.nz/)

## Set up development enviroment
```
bundle install
```

## Run locally
```
bundle exec jekyll serve
```

## Directory structure

| Type     | Path          |
| -------- | ------------- |
| Posts    | `/_posts/`    |
| Projects | `/_projects/` |
| Images   | `/images/`    |
